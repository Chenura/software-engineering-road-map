# 🚀 Software Engineering Roadmap Website   https://chenura.github.io/software-engineering-road-map/

An interactive and modern roadmap website for learning Software Engineering paths including:

- Frontend Development
- Backend Development
- DevOps Engineering
- AI Engineering
- Cybersecurity
- Software Engineering Fundamentals

Built using pure HTML, CSS, and JavaScript with a futuristic developer-themed UI.

---

# ✨ Features

✅ Interactive roadmap cards  
✅ Search functionality  
✅ Responsive design  
✅ Modern futuristic UI  
✅ Beginner to advanced learning paths  
✅ Developer-focused design  
✅ Fully static website  
✅ Fast and lightweight
